# K2E Local v1.7.2 — Offline Charts and Clarity Polish

## What changed

- Chart.js 4.4.0 is now stored locally in `assets/chart.umd.js`.
- The service worker caches the chart library for offline use.
- Visible advisor language now describes the current feature as a local structured advisor rather than implying an external AI service.
- The landing page has a compact local/privacy trust line.
- Confidence-meter guidance now explains which inputs improve estimate quality.

## What did not change

- Energy calculations
- Simple and Advanced mode behavior
- Themes and chart controls
- Saved homes, scenarios, local storage, or imports/exports
- Existing device and schedule data

## Deployment

Upload the contents of this folder to the repository root and commit to the deployment branch.
