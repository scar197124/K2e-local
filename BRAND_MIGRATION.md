# K2E Local Brand Migration

K2E Local is the new public identity for the former MyVera/NRG home energy simulator.

## Public identity

- Brand: **K2E Local**
- Tagline: **Kilowatts to Efficiency**
- Repository recommendation: `K2E-Local`
- Page title: `K2E Local — Home Energy Simulator`

## Data compatibility

Legacy `myvera-*` browser storage keys remain in place intentionally. Changing them would disconnect existing users from saved homes, scenarios, preferences, and chart settings. The keys are implementation details and are not displayed as the public brand.
